# You-give-point-I-give-function

## 功能
这个工具可以通过两点坐标求出它们所在的一次函数表达式。

## 使用
直接git下来之后执行
```command
python main.py
```
就可以，工具有GUI界面。

或者下载最新[Release](https://github.com/zymzs/You-give-point-I-give-function/releases)后打开可执行文件即可使用（目前Release仅发布Windows平台的工具）。
